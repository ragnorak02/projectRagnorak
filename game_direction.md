# Game Direction: Project Ragnorak

## Identity

Working Title: Project Ragnorak
Genre: Korean Fantasy Hybrid Action RPG
Engine: Godot 4.6
Camera: Freely rotatable third-person
Platform: PC
Primary Input: Xbox Controller

## Combat Inspiration

Final Fantasy VII Remake style hybrid combat:
- Real-time movement and attacks
- Tactical slow-time menu (10% speed)
- ATB-based command execution
- Party expansion later

## Core Game Loop

Town hub
-> Explore connected zones
-> Fight enemies in real-time combat
-> Obtain gear and materials
-> Complete quests
-> Unlock traversal abilities
-> Access new areas
-> Repeat

## Core Pillars

1. Controller-first gameplay feel
2. Satisfying hybrid tactical combat
3. Exploration through traversal progression
4. System stability over feature speed
5. Deterministic, bug-resistant architecture

## World Design

- Semi-open connected zones
- Multiple towns
- Handcrafted dungeons with light puzzles
- Hidden areas gated by traversal abilities
- Loading transitions between major regions

## Character Design

- Protagonist with fixed story identity
- Party members join through story progression
- AI companions control inactive members
- Character switching in and out of battle
- Equipment visually changes character appearance

## Combat Design

- No stamina system, basic attacks unlimited
- Spells use MP, tactical commands require ATB
- Dodge is timing-based with NO invulnerability frames
- Basic attacks cancel into dodge, spells do not
- Enemies can interrupt spell casting
- Lock-on prefers nearest enemy with camera shift
- Jump attacks available from early development

## Art Direction

- Korean fantasy aesthetic
- Dark but vibrant color palette
- Placeholder assets during development
- Equipment visual changes planned for later phases
